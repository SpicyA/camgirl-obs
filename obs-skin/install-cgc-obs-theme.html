CGC OBS Theme

About

CGC OBS theme is NOT an OBS overlay or something like that, with this package installed, the OBS interface changes colors, a few buttons and icons.

CGC is based on Dark (default) OBS theme but with some changes.

Install and Use

Install process is simple, just follow these steps:

  1. Download and unzip the content of the zip file
  2. Locate the theme directory of your OBS, for me is default C: partition on my windows OS
    C:\Program Files\obs-studio\data\obs-studio\themes
  3. Copy the content of the obs_theme folder in your OBS's theme's directory. There's a folder and a .qss file, both need to be placed in themes
    CGC (folder) CGC.qss (file)
  4. Open OBS, go to Settings » General » from drop-down box of Themes choose the CGC » click Apply button

That's it

I didn't use plugins or additional scripts so the CGC theme is not tested with other addons on OBS but it should work ok
